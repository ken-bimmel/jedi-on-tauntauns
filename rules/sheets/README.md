# Sheets

There are currently two sheets currently available:

1. The [Character sheet](./Jedi%20on%20Tauntauns%20Character%20Sheet.pdf)
2. The [Vehicle sheet](./Jedi%20on%20Tauntauns%20Vehicle%20Sheet.pdf)

## Modifying the sheets

All sheet graphics were created using [Excalidraw](https://excalidraw.com/). The
SVGs in this folder can be opened using Excalidraw and modified there or
modified by hand.

The PDFs are manually created from the SVG files for that particular sheet.

**Note:** You may need to install the Excalidraw font,
[Excalifont](https://plus.excalidraw.com/excalifont), in order to properly
render the SVGs for printing.
